# Documentation

이 폴더는 프로젝트 관련 문서를 관리합니다.

## 폴더 구조

```
docs/
├── plans/          # 작업 계획 및 요구사항 문서
│   └── emrcert.md  # EMR 인증 크롤링 작업 계획
├── guides/         # 사용 가이드 (향후 추가)
└── api/            # API 문서 (향후 추가)
```

## plans/ 폴더

각 크롤링 프로젝트의 작업 계획서를 저장합니다.

### 작성 내용
- 프로젝트 개요 및 목표
- 대상 웹사이트 정보
- 수집할 데이터 필드
- 기술적 고려사항
- 예상 소요 시간

### 파일 명명 규칙
- `{project_name}.md` 형식으로 작성
- 예: `emrcert.md`, `hospital_data.md`
